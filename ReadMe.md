## DL Sprint Repository

This repository contains notebooks and scripts for the following Kaggle competitions, focused on audio processing, segmentation, diarization, and ASR fine-tuning:

- [DL Sprint 4.0 Bengali Long Form Speech Recognition](https://www.kaggle.com/competitions/dl-sprint-4-0-bengali-long-form-speech-recognition)
- [DL Sprint 4.0 Bengali Speaker Diarization Challenge](https://www.kaggle.com/competitions/dl-sprint-4-0-bengali-speaker-diarization-challenge)

### Structure
- Inference Notebooks/: Jupyter notebooks for inference pipelines, diarization, and correction tasks.
- tuning/: Notebooks for fine-tuning segmentation and ASR models, hyperparameter tuning, and VAD diarization.
- post processing/: Python scripts for cleaning and post-processing diarization outputs.

### Usage
Open the relevant notebook in Jupyter to run experiments or inference. Use scripts in post processing for output cleanup.

For details, see individual notebook and script documentation.
